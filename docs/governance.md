# Governance

## Ownership

- Product area: Laboratory and Technical Services (FEIT)
- Repository maintainers: to be assigned

## Review Cadence

- Service definitions should be reviewed at least once per semester.
- Critical services should be reviewed after major operational changes.

## Pull Request Requirements

- Service owner reviewed
- Required fields completed
- Contact details validated
- Lead time and availability confirmed

## Versioning

- Catalogue entries are version-controlled via Git history.
- Significant service changes should be documented in pull request descriptions.
